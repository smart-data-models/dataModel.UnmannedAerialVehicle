Descrizione globale: **Questa entità contiene una descrizione armonizzata di un generico UAV UTM Flight Message Agent che è progettato per sottoscrivere il messaggio del protocollo Global UTM Association secondo una specifica entità UAV. Questa entità supporta la funzionalità di un fornitore di servizi per confermare la validità dell'UTM Flight Message generato dall'UTM Flight Message Entity. Il fornitore di servizi può includere la propria politica di controllo del volo al messaggio di volo UTM originale e inoltrarlo a un'entità UAVTMS.**
